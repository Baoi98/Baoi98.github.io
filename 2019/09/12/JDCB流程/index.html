发布于 2019-09-12   更新于 2019-09-12 Java 面试题

JDBC流程

  1. 向 DriverManager 类注册驱动数据库的驱动程序。
  2. 调用 DriverManager.getConnection() 方法，通过 JDBC URL、用户名、密码获取数据库连接。
  3. 获取 Connection 后，创建 prepareStatement（预编译，推荐使用）或 Statement（不推荐）对象执行SQL语句。
  4. 有时候会得到查询结果，查询的结果存放在结果集（ResultSet）中。
  5. 关闭数据库语句（prepareStatement对象），关闭数据库连接。
