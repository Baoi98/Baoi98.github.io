SpringMVC文件上传

2019-08-27 Spring PV:


SpringMVC 为上传文件提供了良好的支持，SpringMVC 通过 MultipartResolver (Multipart 解析器) 处理的，它是一个接口，有两个实现类:
​

  • CommonsMultipartResolver：依赖于 Apache 下的 jakarta Common FileUpload 项目解析 Multipart 请求，可以在 Spring 的各个版本中使用，只是它要依赖于第三方包才得以实现
  • StandardServletMultipartResolver：Spring 3.1 版本后的产物，它依赖于 Servlet 3.0 或者更高版本的实现，它不用依赖第三包
